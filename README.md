# 🌱 FarmPal — Smart Agriculture for Every Farmer

FarmPal is a zero-dependency, high-performance web application designed to empower smallhouse farmers with real-time data and AI-driven insights. 

## ✨ Key Features
- **📊 Interactive Dashboard**: Real-time farm telemetry and health overview.
- **🚜 Smart Crop Management**: Track lifecycle stages from planting to harvest.
- **⛅ Weather Integration**: Localized forecasting and crop-specific alerts.
- **🤖 AI Recommendations**: Rule-based engine providing actionable agricultural tips.
- **📱 PWA Ready**: Designed for offline use in low-connectivity rural environments.

## 🚀 Demo
The live demo is hosted on [GitHub Pages](https://github.com/YourUsername/farmpal).

## 🛠 Tech Stack
- **HTML5/CSS3**: Custom dark-mode design with Glassmorphism.
- **Vanilla JavaScript**: High performance, zero external browser dependencies.
- **LocalFirst**: Built-in persistence for offline reliability.
